The Robot

This is our robot, the Ruumba. It uses Pneumatics to apply air pressure so that the clamp can clamp down upon mogos,
then it will lift them making it easy for us to move them around the rink.
Our main goal is to bring back mogos to our side or lifting them up onto the alliance platform to score more points.
We also plan to create a system which can pick up rings that would be then transfered into a mogo, thus gaing more points.
When Automnous starts we must solely start on the left side because right now we only have the code for left,
but Hayden Maus has been working on a program which can work for the right side.
